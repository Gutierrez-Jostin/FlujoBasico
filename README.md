# Practica 2 - Preguntas

1.¿Qué es una Base De Datos Relacional?
  R= Es un tipo de base de datos que organiza la informacion en tablas estructuradas (filas y columnas) con relaciones definidas entre ellas

2.¿Qué es una Tabla en el contexto Base De Datos Relacionales?
  R= Es la estructura fundamental organizada en filas (registros) y columnas (campos) que almacena datos sobre una entidad especifica

3.¿Qué es una relación en el contexto de Base De Datos Relacionales?
  = Es una conexion logica entre dos o mas tablas, establecidas mendiente campos comunes (claves principales y foraneas) para organizar datos, garantizar la integridad y evitar duplicidades
4.¿Qué representa una columna en el contexto de Base De Datos Relacionales?
  R= Representa una caracteristica, propiedad o atributo especifico de la entidad que se esta modelando en una tabla

5.¿Qué representa un renglón en el contexto de Base De Datos Relacionales?
  R= Un registro individual, unico y especifico dentro de una tabla

6.¿Qué es un motor de base de datos relacional?
  R= Es un sistema de software que almacena, destiona y recupera datos organizados en tablas interconectadas (filas y columnas), garantizando integridad y consistencia mediante el lenguaje SQL

7.¿Cuáles son los motores de bases de datos relacionales mas populares?
  R= Son MySQL, PostgreSQL, Microsoft SQL Server y Orancle Database

8.¿Cuál es el motor de base datos ligero que no necesita un servidor?
  R= El motor de base de datos ligero, autonomo y sin servidor mas popular es SQLite

9.¿Qué significa SQL en el contexto de base datos relacionales?
  R= Structured Query Language (Lenguaje de Consulta Estructurado)

10.¿Cuáles son los elementos de un sistema de información basado en base de datos relacionales?
   R= Son tablas, claves (primarias/foraneas), consultas (SQL), el SGBDR (sistema de gestion) y reglas de integridad
